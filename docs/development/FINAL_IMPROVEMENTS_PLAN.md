# План фінальних покращень

## 1. Виправити перемикання gateway
**Проблема:** Завжди підключається до першого gateway  
**Причина:** Використовується `connect(portal)` замість `connectToGateway(gateway)`  
**Рішення:** Змінити на `connectToGateway(gateway)` після `setPreferredGateway`

## 2. Кешування списку gateway
**Проблема:** Список запитується кожен раз при відкритті меню  
**Рішення:** Кешувати список при першому запиті, оновлювати тільки при зміні підключення

## 3. Кешування details після підключення
**Проблема:** Details запитуються кожен раз  
**Рішення:** Запитувати details після успішного підключення, зберігати в statusMonitor

## 4. Показувати Gateway info в статусі
**Проблема:** Статус показує тільки "Connected to portal"  
**Рішення:** Додати Gateway Name, Assigned IP, Gateway IP після "Connected"

## 5. Прибрати Show → Connection Details
**Рішення:** Видалити з меню (інфо вже в статусі)

## 6. Виправити Collect Logs
**Проблема:** Не відкриває теку  
**Рішення:** Перевірити код відкриття теки

## 7. Вирівняти Advanced меню
**Рішення:** Додати відступ для вирівнювання

## 8. Додати Show → Host State
**Рішення:** Додати новий пункт меню з `show --host-state`
